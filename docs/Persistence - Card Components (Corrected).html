Persistence · Handoff supplement

Card components, corrected to the prototype.

Three composite cards drifted in the React-Native port because they were never documented as primitives — they live only inside home.jsx as screen-local functions. This sheet pins each one to its exact prototype spec. Match these values; they are the source of truth.

source · design-source/screens/home.jsx tokens · theme.css viewport · 375–402px
Why these drifted

The design-system doc documents base primitives (Card, Pill, Btn, Avatar, rings, bars) and a standalone gold "stat card" demo — which is not the PR carousel tile. With no contract for the composite cards, the port reinvented them. Build each as a presenter that composes the existing primitives; do not restyle the primitives themselves.
